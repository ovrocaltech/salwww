OVRO Software & Algorithms Lab

Software is increasingly critical to meeting the scientific goals in astronomical research. New instruments produce more and more diverse kinds of data. The Software and Algorithms Lab is dedicated to developing software instrumentation and tools in support of OVRO projects.

For more information, contact Software and Algorithms Lead, Dr. Casey Law.

The OVRO Software Group facilitates discussion and development of software in topics such as:

  • Calibration, imaging and algorithmic development
  • Data and archives
  • Microservices
  • Software development practices
  • Monitor and control systems
  • Real-time processing
  • Machine learning
  • Computing platforms (CPUs, GPUs, FPGAs, cloud)
